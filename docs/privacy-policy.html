개인정보처리방침

최종 업데이트: 2026년 3월 27일

UJAX Problem Collector(이하 "확장 프로그램")는 알고리즘 스터디 플랫폼 UJAX와 연동하여 백준 온라인 저지의 문제 정보 및 제출 결과를 자동으로 수집·기록하는 Chrome 확장 프로그램입니다. 본 방침은 확장 프로그램이 수집하는 데이터와 그 사용 방식을 설명합니다.

1. 수집하는 데이터

  • UJAX 인증 토큰(JWT) — UJAX 로그인 후 브라우저 localStorage에 저장된 액세스 토큰을 읽어 확장 프로그램 내부 저장소(chrome.storage.local)에 보관합니다. UJAX 서버에 문제·제출 데이터를 전송할 때 인증 수단으로만 사용됩니다.
  • 백준 아이디(ID) — 백준 온라인 저지 페이지에서 로그인된 사용자 아이디를 감지하여 UJAX 계정과 자동 연동하는 데 사용됩니다.
  • 문제 메타데이터 — 백준 문제 페이지 방문 시 문제 번호, 제목, 설명, 입출력 예제 등을 수집하여 UJAX 서버에 등록합니다.
  • 제출 결과 — 백준 채점 현황에서 감지된 제출 ID, 결과(정답/오답 등), 언어, 소스 코드를 UJAX 서버에 기록합니다.
  • 수집 이력(로컬 캐시) — 중복 수집 방지를 위해 수집된 문제 번호 및 전송된 제출 ID 목록을 로컬에 저장합니다.

2. 데이터 사용 목적

수집된 데이터는 오직 아래 목적으로만 사용됩니다.

  • 백준 문제 정보를 UJAX 플랫폼에 등록
  • 사용자의 제출 결과를 UJAX 스터디 기록에 저장
  • 백준 아이디와 UJAX 계정 자동 연동

3. 데이터 공유 및 제3자 전송

수집된 데이터는 사용자가 직접 가입한 UJAX 서버(ujax.kro.kr)에만 전송됩니다. 광고, 분석, 마케팅 등 다른 목적으로 제3자에게 판매하거나 전송하지 않습니다.

문제 난이도 및 태그 정보 보강을 위해 solved.ac 공개 API를 호출합니다. 이 과정에서 문제 번호만 전송되며 개인 정보는 포함되지 않습니다.

4. 데이터 보관 및 삭제

모든 데이터는 사용자의 브라우저 로컬 저장소(chrome.storage.local)에 보관됩니다. 확장 프로그램을 삭제하면 모든 로컬 데이터가 함께 삭제됩니다. UJAX 서버에 전송된 데이터는 UJAX 서비스 약관에 따라 관리됩니다.

5. 원격 코드 사용

확장 프로그램은 외부에서 코드를 다운로드하거나 eval()을 사용하지 않습니다. 모든 코드는 확장 프로그램 패키지 내에 포함되어 있습니다.
